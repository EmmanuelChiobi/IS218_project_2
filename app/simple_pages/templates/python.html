What is Pytest?

Pytest is a mature full-featured Python testing tool that helps you write better programs. It is a testing framework that allows users to write test codes using the Python programming language. It helps you to write simple and scalable test cases for databases, APIs, or UI.

What is Flask?

A lightweight WSGI web framework, Flask is a Python module that lets you develop web applications with ease. It is designed to make getting started with web development quick and easy with the ability to scale up to complex applications.

How They're Used In Our Code

Regarding Flask, it is essentially the backbone, or infrastructure, of the entire project running on a virtual environment as the Python interpreter. Regarding Pytest, the tool is divided into various files each with different responsibilities for the Flask project. For this project in particular, two files that are of importance include the "context_process_text.py" and "simple_pages_text.py" files, which are essentially for the testing of the individual pages for this web application project.

When creating and testing "simple_pages_test.py", I had to make a few edits to the program in order to adapt with the new title change of the simple pages. For instance, they were initially called "page1, page2, etc.", so when I changed them to the respective topics being covered for this project, I also had to change the code to reflect on the new titles of the html files in the simple pages directory. After that, I created a build using the "docker compose up --build" command, launched Docker from within PyCharm, and conducted the tests in the terminal only to see that they were a success.
